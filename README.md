# VOICE LAB v7.0 — 声質診断システム

マイクで声を録音するだけで、音響特性を計測し、248名の声データベースからあなたの声に近い声優・歌手・俳優・アナウンサーをマッチングするWebアプリケーションです。


## 機能概要

### 計測する音響特性
| 指標 | 説明 |
|------|------|
| 基本周波数 F0 | 声の高さ（Hz）。自己相関法で推定 |
| 最低 / 平均 / 最高音域 | 録音中の音域レンジを3点マーカーで可視化 |
| ピッチ安定性 | F0の変動係数から算出（0〜100%） |
| 調波雑音比 HNR | 倍音ピークと雑音フロアの比（声の澄み具合） |
| スペクトル重心 | 音色の明るさを示す加重平均周波数 |
| 低域エネルギー比 | 450Hz以下の声の温かみを示す割合 |
| 高域成分比率 | 2500Hz以上の輝きを示す割合 |

### マッチングアルゴリズム
声データベース248名に対して以下のスコアで順位付けし、ジャンル別上位5名を表示します。

```
スコア = 音域重複率（最大52点）
       + 平均F0距離（最大22点）
       + 音色一致（温かみ+明るさ、最大24点）
```

### 声データベース構成
- **男性**: バス〜ハイテノール（5帯域）
- **女性**: コントラルト〜ソプラノ（4帯域）
- **ジャンル**: 声優 / 歌手・アーティスト / 俳優・タレント / アナウンサー・ナレーター

## ホスティング方法

### 静的ホスティング（推奨）
このパッケージは完全に静的なファイルで構成されています。サーバーサイドの処理は不要です。

**主要なホスティングサービス例：**
- **Netlify** — フォルダをドラッグ&ドロップするだけでデプロイ可能
- **Vercel** — GitHubリポジトリと連携して自動デプロイ
- **GitHub Pages** — リポジトリの `main` ブランチから直接公開
- **Cloudflare Pages** — 高速CDNで世界配信

### ローカル確認方法
ブラウザのセキュリティポリシーにより、`file://` で直接開くとマイクAPIが動作しないことがあります。ローカルで確認する場合はHTTPサーバーを使ってください。

```bash
# Python 3
python3 -m http.server 8080

# Node.js (npx)
npx serve .
```

ブラウザで `http://localhost:8080` を開きます。

## プライバシー

録音した音声データはブラウザ内でのみ処理されます。音声データをサーバーに送信することは一切ありません。

---

## 注意事項

- WebAudio APIによる近似計測のため、専門ツール（Praat等）の数値とは異なる場合があります
- 「いい声」の感覚的評価には個人差・文化差があります。本結果は音響的傾向の参考情報として活用してください
- 静かな環境でマイクに近づいて録音すると、より正確な計測結果が得られます
